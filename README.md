# Mongo backup agent
An open source mongo backup agent for backing up an existing mongo db to another mongo db. 

## Getting Started
### Install
```node
npm install  
```
### Running the project
On dev environment:
```
npm run dev
```
On prod environment: 
```
npm run start
```

## Tech/framework used
* NodeJS
* Mongodb
* Typescript
